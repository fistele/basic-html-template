# Projet Semaine 1 : Initiation au HTML5
Ce projet est une introduction aux bases du HTML5. L'objectif est de créer une structure de page web simple en utilisant des balises HTML5 de base et des éléments multimédias.

## Objectifs
- Comprendre les balises HTML5 de base (titres, paragraphes, listes, liens, images).
- Créer une structure de page web avec un header, un main et un footer.
- Intégrer des éléments multimédias (image, vidéo).
- Explorer les balises sémantiques (section, article, nav, etc.).

## Structure du projet
- `index.html` : La page web principale avec une structure HTML5 simple.
- `README.md` : Ce fichier, qui explique le projet.

## Comment utiliser
1. Clonez ce dépôt sur votre machine.
2. Ouvrez le fichier `index.html` dans un navigateur pour voir la page web.
3. Explorez le code pour comprendre comment les balises HTML5 sont utilisées.

## Exemples de balises utilisées
- `<header>`, `<main>`, `<footer>` pour la structure de la page.
- `<section>` pour organiser le contenu.
- `<img>` et `<video>` pour les éléments multimédias.
- `<nav>` pour la navigation.

## Tâches facultatives
- Ajoutez d'autres balises sémantiques comme `<article>` ou `<aside>`.
